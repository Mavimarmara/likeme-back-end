# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2024-01-XX

### Adicionado
- Sistema completo de autenticação com JWT
- CRUD de usuários com validação
- Sistema de anamnese com questionários personalizados
- Gerenciamento de atividades de saúde
- Dashboard de bem-estar com métricas
- Sistema de comunidade com posts, likes e comentários
- Marketplace com produtos e pedidos
- Sistema de provedores de saúde e agendamentos
- Documentação completa da API com Swagger
- Testes automatizados
- Docker e Docker Compose para desenvolvimento
- CI/CD com GitHub Actions
- Rate limiting e segurança
- Validação de dados com Joi
- Logging e monitoramento
- Estrutura de banco de dados com Prisma ORM
- Seed de dados iniciais
- Scripts de setup para desenvolvimento e teste
